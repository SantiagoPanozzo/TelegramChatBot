Detailed Description

Muestra una lista de OfertaDeServicio disponibles según su ID y el trabajo. El usuario puede seleccionar un filtro del BuscarHandler para cambiar el orden en el que se muestran las ofertas. Puede elegir una oferta en concreto escribiendo su ID y esto le dará toda la información de la misma, en lo cual podrá crear una solicitud para esa oferta o volver a la lista.

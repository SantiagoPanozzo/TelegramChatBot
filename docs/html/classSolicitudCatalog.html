Detailed Description

Clase para para manejar el catálogo de solicitudes.

Utilizamos patrón singleton ya que solo necesitamos una misma instancia de esta clase, si hubieran más se mezclarían los elementos de la misma y no sabríamos a cual instancia acceder para obtener las solicitudes.
